NAME

Archive::Tyd::Checksum - Base class for checksum handlers.

SYNOPSIS

  package Archive::Tyd::Checksum::MyChecksum;
  use base qw(Archive::Tyd::Checksum);

DESCRIPTION

This is the base class for checksum handlers for the Archive::Tyd manpage. Checksums are used for verifying the integrity of a Tyd archive or for individual members of the archive.

METHODS

You must override the following methods.

string name ()

This should return the algorithm's name, e.g. SHA1.

string digest (filehandle)

This should take a file handle and return a checksum hash based on it.

bool verify (string hash, filehandle)

Verify that the given hash is the correct checksum for the file handle.
