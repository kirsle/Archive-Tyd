NAME

Archive::Tyd::Algorithm - Base class for file mangling algorithms.

SYNOPSIS

  package Archive::Tyd::Algorithm::Deflate;
  use base qw(Archive::Tyd::Algorithm);

DESCRIPTION

This is the base class for creating custom file mangling algorithms. These algorithms are intended for compression or encryption of the individual files in a Tyd archive.

METHODS

string name ()

This should return the name of the encryption algorithm, preferrably the fully qualified package name (for ex. Archive::Tyd::Algorithm::Deflate).

string provides ()

This should return a list of services provided by the algorithm. Currently the following services are supported:

  encoding: The algorithm simply mangles data (this is useful for symmetric
            key encryption or compression algorithms).
  signing:  This algorithm supports signatures (for example RSA)

void init (args)

If your algorithm requires any initialization (for example, providing of an encryption key) it should handle it in this method. args is a list of arguments provided by the user when they request this algorithm.

bin encode (bin file_contents)

Given the file_contents as a scalar, encode it and return the encoded version. This version should be the compressed or encrypted output.

bin decode (bin encoded)

Given the encoded data, decode it and return the original data.

bin sign (bin data)

Create a signature for the given data. This is only applicable to algorithms that provide signing.

bool verify (big signature, bin data)

Verify the signature for the given data. This is only applicable to algorithms that provide signing.
